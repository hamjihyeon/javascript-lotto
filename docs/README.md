## 사용자
- [ ] 1장 당 1000원
- [ ] 

## 로또 번호
- [ ] 숫자의 범위는 1 ~ 45
- [ ] 랜덤으로 숫자 6개 뽑기
- [ ] 오름차순으로 정렬

## 당첨번호
- [ ] 숫자 7개 입력받기
  - [ ] 중복되지 않아야함
  - [ ] 일반숫자 6 + 보너스 숫자 1
- [ ] 쉼표(,)로 구분

## 당첨
- [ ] 1등: 6개 번호 일치 / 2,000,000,000원
- [ ] 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
- [ ] 3등: 5개 번호 일치 / 1,500,000원
- [ ] 4등: 4개 번호 일치 / 50,000원
- [ ] 5등: 3개 번호 일치 / 5,000원
- [ ] 수익률 보여주기
  - [ ] 소수점 둘째 자리에서 반올림
